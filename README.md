# Fruit Ninja

Fruit Ninja is an engaging and fun project designed for users to experience the thrill of slicing fruits without missing. This repository contains the full codebase and assets required to play and experiment with the game.

## Features

- **Interactive Gameplay**: Slice the fruits as they appear on the screen.
- **Beginner-Friendly**: Easy to understand and perfect for practice.
- **Customizable**: Modify and enhance the gameplay according to your preference.

## Requirements

Before you begin, ensure you have met the following requirements:

- [HTML](https://www.w3schools.com/html/) is the primary language used in this project.
- A modern web browser to run the project.
- No additional installations required.

## Installation and Usage

1. Clone the repository using the following command:

   ```bash
   git clone https://github.com/UmairBaig8/Fruit-ninja.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Fruit-ninja
   ```

3. Open the `index.html` file in your preferred web browser.

4. Start slicing fruits!
